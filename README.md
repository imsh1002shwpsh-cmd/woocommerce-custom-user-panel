# WooCommerce Custom User Panel

پنل کاربری سفارشی و سیستم ورود/ثبت‌نام پیشرفته برای WooCommerce

## ویژگی‌های اصلی

### سیستم ورود و عضویت
- ✅ ورود/ثبت نام با شماره موبایل
- ✅ ورود/ثبت نام با ایمیل
- ✅ ورود با رمزعبور
- ✅ تایید OTP برای موبایل و ایمیل
- ✅ امنیت کامل

### پنل کاربری
- ✅ طراحی ریسپانسیو
- ✅ مدیریت پروفایل
- ✅ بارگزاری تصویر پروفایل
- ✅ مشاهده سفارشات
- ✅ سیستم تیکتینگ
- ✅ لیست علاقه‌مندی‌ها
- ✅ بخش اطلاعیه‌ها
- ✅ کدهای تخفیف

### پنل مدیریت
- ✅ مدیریت اطلاعیه‌ها
- ✅ مدیریت کدهای تخفیف
- ✅ مدیریت تیکت‌های پشتیبانی
- ✅ تنظیمات عمومی

## نیازمندی‌ها

- WordPress 5.0+
- PHP 7.4+
- WooCommerce 4.0+

## نصب

1. افزونه را در مجلد `wp-content/plugins/` قرار دهید
2. افزونه را فعال کنید
3. به تنظیمات User Panel بروید
